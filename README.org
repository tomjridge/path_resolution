Simple specification/implementation of core path resolution, taking
account of real-world behaviour uncovered by SibylFS.

https://tomjridge.github.io/path_resolution/path_resolution.ml.html

https://tomjridge.github.io/path_resolution/tjr_path_resolution.mli.html


To build, type `make`

NOTE this installs a package `tjr_path_resolution` into ocamlfind/opam


For documentation, please read the source code of `path_resolution.ml`
